Visualizing Incident Analytics for Enhanced Monitoring

Description
Feature Overview This feature allows users to visualize incident data over time and analyze key concerns with interactive and visually appealing charts.
Incidents Over Time Chart This chart displays the number of incidents recorded each month, helping users identify trends and patterns. The bar chart utilizes soft, transparent cyan tones with a darker hue on hover to ensure an engaging and clear visualization.
Activity Concerns Pie Chart This pie chart breaks down reported concerns into categories (e.g., safety, infrastructure, traffic). Each category is represented by a soft, pastel color palette with smooth transitions and hover effects, enabling users to quickly understand the distribution of concerns.
Summary Both charts are designed with an emphasis on clarity and aesthetic appeal, ensuring users can monitor and analyze incident data effectively.
