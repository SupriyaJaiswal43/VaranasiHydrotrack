Overview

This is designed to visualize water quality data using charts. It provides functionality for users to select different sites, choose various chart types, filter data by year, and download charts individually or as a PDF. The dashboard also supports real-time updates to ensure that the data remains current.

Data Fetching

  • Functionality: Fetches water quality data from a local server API.
  • Details: Handles errors and processes the data, converting time periods to years and setting default values for missing parameters.

Site Selector

  • Functionality: Allows users to select a site from a dropdown menu.
  • Details: Populates the dropdown with unique site names extracted from the data. Updates charts based on the selected site.

Chart Type Selector

  • Functionality: Lets users choose the type of chart to display.
  • Options: Line Chart, Bar Chart, Pie Chart, Doughnut, Polar Area Chart, Radar Chart.
  • Details: Updates the displayed chart according to the selected type.

Year Filter

  • Functionality: Filters data by a specific year.
  • Details: When the "Filter by Year" button is clicked, it filters the data based on the input year and updates the charts.

Chart Creation

  • Functionality: Generates charts for visualizing the water quality parameters.
  • Details: Uses Chart.js to create various types of charts with customizable options like labels and data. Each parameter (e.g., pH, TDS, Turbidity) is represented with distinct colors.

Chart Update

  • Functionality: Updates the charts displayed in the container based on the selected site or filter.
  • Details: Clears existing charts and creates new ones based on the filtered or selected data.

Download Chart as Image

  • Functionality: Allows users to download individual charts as PNG images.
  • Details: Uses a function to convert chart canvases to image data URLs and triggers a download.

Generate PDF Report

  • Functionality: Generates a PDF report with the current charts and data.
  • Details: Collects chart images, combines them with data, and creates a downloadable PDF file.

Interactive Website

  • Functionality: Provides an interactive Website for exploring and analyzing water quality data.
  • Details: Features real-time data updates, customizable chart options, and user-friendly navigation.
